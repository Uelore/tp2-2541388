# Travail Pratique 2
Ce dépôt contient le code d’un projet web fait dans le cours de Perspective Professionnelles.
### Faire un commit et un push dans Visual Studio Code
Voici comment faire un changement sur ce projet, et l’envoyer sur GitHub.
1. Aller cherchez les dernières modifications.
2. Faire un modification ou un ajout au projet.
3. Donner un nom au commit et appuyer sur le bouton bleu pour envoyer les modifications.
4. Appuyer sur le bouton bleu pour faire un push!
